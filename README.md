# Bird Bingo Application

This application was developed for the **Programming 2** course (Autumn 2024).

## Overview
Bird Bingo is an object-oriented Java application designed for recording bird sightings.  
The program allows users to store, manage, and review observed birds.

## Features
- Add new bird sightings
- Store bird data
- View recorded observations

## Technologies
- Java
- Object-Oriented Programming principles
